## 🚀 核心优先级 (Top Priorities)

1.  **全站视觉标准化**：彻底清理粉色系残留，统一使用金色主题变量，移除所有 Emoji 图标。
2.  **管理后台真实化**：将管理后台从模拟数据转向真实 API 对接，补齐缺失的管理页面。
3.  **算法准确性升级**：提升八字强弱判断准确度，完善六爻变卦、六亲、六神分析系统。
4.  **系统安全加固**：修复 SQL 注入风险，移除硬编码凭据，完善 API 响应一致性。

## 💻 后端逻辑与系统安全

### 🔴 高优先级
- [ ] **安全风险修复**：
    - 修复 `Admin.php` 和 `Content.php` 中的 SQL 注入隐患，使用参数绑定替代字符串拼接。
    - 移除 `AdminAuth.php` 中的硬编码管理员凭据，对接数据库验证。
    - 将 JWT 密钥等敏感信息移出代码，改用环境变量读取。
- [ ] **API 规范化**：
    - 统一后端响应码，成功一律返回 `200`（目前部分接口返回 `0`）。
    - 统一使用 `BaseController` 的 `success()` 和 `error()` 方法返回数据。
- [ ] **功能补全**：
    - 实现 Dashboard 实时统计数据接口 (`/dashboard/realtime`)。
    - 实现待处理反馈统计接口 (`/dashboard/pending-feedback`)。

### 🟡 中优先级
- [ ] **代码质量优化**：
    - 提取各控制器中重复的分页验证和权限检查逻辑到 `BaseController` 或 `Trait`。
    - 为流式响应 (SSE) 添加更完善的错误处理和超时机制。
    - 完善前端模板中的可选链操作符 (`?.`)，防止因后端数据缺失导致的页面崩溃。
- [ ] **输入校验加固**：
    - 完善 XSS 过滤（推荐引入 `DOMPurify`）。
    - 为所有 API 请求添加严格的参数类型和长度验证。
