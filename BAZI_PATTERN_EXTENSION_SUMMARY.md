
### 核心算法

#### 格局识别算法
1. 获取月令十神
2. 检查日主强弱
3. 验证格局条件
4. 评估格局质量
5. 生成格局描述

#### 神煞识别算法
1. 根据日主查找神煞地支
2. 匹配四柱地支
3. 识别神煞位置
4. 评估神煞质量
5. 生成神煞描述

#### 大运流年分析算法
1. 重新计算五行统计
2. 重新计算十神
3. 检测格局变化
4. 喜用神加分/减分
5. 生成评分和建议

#### 运势评分算法
```
总评分 = 基础评分 + 吉神加分 - 凶神减分

基础评分 = 格局层次评分
吉神加分 = Σ(吉神等级 × 权重)
凶神减分 = Σ(凶煞等级 × 权重)
```
