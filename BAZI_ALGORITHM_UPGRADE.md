# 八字算法升级说明

## 📋 升级概述

**升级日期**: 2026-03-20
**升级内容**: 八字算法全面升级，增加格局分析和命理定语功能
**目标**: 提升八字测算的准确性和解读深度

---

## 🎯 核心改进

### 1. 新增格局分析服务

创建了全新的 `BaziPatternService` 服务类，包含以下功能：

#### 八格分析（正格）
- **正官格**: 月令正官为提纲，性格正直守信，重视名誉，事业心强
- **七杀格**: 包含多种形态
  - 食神制杀格（配印）：上上格，威猛有制有化，刚柔并济
  - 食神制杀格：上格，七杀得食神制伏，威猛有制
  - 杀印相生格：上格，七杀得印星化解，威猛转为文治
  - 七杀格（无制化）：偏格，性格急躁，压力大
- **正财格**: 勤俭持家，务实稳重，重视承诺
- **偏财格**: 善于理财，人缘好，慷慨大方
- **正印格**: 心地善良，有学识，易得贵人帮助
- **食神格**: 聪明有才华，善于表达，有口福

#### 特殊格局
- **三合局**: 申子辰、亥卯未、寅午戌、巳酉丑
- **三会方**: 寅卯辰、巳午未、申酉戌、亥子丑
- **从格**: 
  - 假从格：日主极弱，其他五行极强
  - 真从格：日主无根，其他五行极强

#### 组合格局
- **官印相生格**: 上上格，有事业心且有学识，能够得到贵人相助
- **伤官驾杀格**: 上格，才华与魄力并存
- **食伤生财格**: 上格，才华能够转化为财富
- **比劫帮身格**: 中格，能够得到朋友和兄弟姐妹的帮助

#### 神煞分析
- **天乙贵人**: 一生有贵人相助，逢凶化吉，遇难呈祥
- **文昌**: 聪明有才华，善于学习，有考试运
- **驿马**: 一生奔波劳碌，见识广博，经验丰富
- **桃花**: 异性缘好，魅力十足，感情运势佳
- **羊刃**: 性格刚烈，有魄力，但也容易冲动

#### 命理定语生成
根据八字格局、神煞、五行等因素，自动生成个性化的命理定语：
- 日主强弱定语
- 格局定语
- 神煞定语
- 五行定语

#### 格局层次评估
将格局分为五档：
- **上上格**: 格局层次很高，富贵可期
- **上格**: 格局层次较高，事业有成
- **中上格**: 格局层次中上，事业顺遂
- **中格**: 格局层次中等，事业平稳
- **下格**: 格局层次较低，需要更多努力

## 🔧 技术实现细节

### 格局分析算法

#### 八格识别流程
1. 获取月令十神
2. 检查日主强弱
3. 验证格局条件
4. 评估格局质量
5. 生成格局描述

#### 特殊格局识别
1. 检查三合局（申子辰等）
2. 检查三会方（寅卯辰等）
3. 检查从格条件
4. 评估格局层次

#### 神煞识别
1. 根据日主查找天乙贵人地支
2. 根据日主查找文昌地支
3. 根据日主查找驿马地支
4. 根据日主查找桃花地支
5. 根据日主查找羊刃地支

### 命理定语生成逻辑

```php
// 命理定语 = 日主强弱 + 格局 + 神煞 + 五行
$mingliDingyu = [
    $dayMasterStrengthDescription,  // 日主强弱定语
    $mainPatternDescription,         // 格局定语
    $topShenShaDescription,          // 神煞定语
    $dominantWuxingDescription,      // 五行定语
];
```
