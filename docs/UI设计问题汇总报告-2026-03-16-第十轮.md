## 🔴 高优先级问题（功能性问题）

### 1. 主题方向决策（最核心问题）

**问题描述**:  
style.css定义了白色主题（--bg-primary: #ffffff），但所有页面实际使用深色背景（rgba(0,0,0,0.2)、rgba(255,255,255,0.05)）和白色文字。这是最核心的设计问题，影响整个网站的视觉一致性。

**影响范围**: 全站所有页面

**建议方案**:  
考虑到命理玄学的行业属性，建议评估以下两个方向：
- **方案A：深色主题** - 神秘、专业感，更符合命理玄学调性
- **方案B：白色主题** - 清新、现代感，更符合当前设计趋势

**决策后行动**:
1. 统一修改style.css中的CSS变量
2. 批量更新所有页面的背景色和文字颜色
3. 更新Element Plus主题配置

---

### 2. 导航栏与页面内容区视觉割裂

**问题描述**:  
App.vue中导航栏使用白色主题（rgba(255,255,255,0.95)），但各页面内容区使用深色背景，造成严重的视觉割裂感。

**具体位置**: App.vue第295-304行

```css
.navbar {
  background: rgba(255, 255, 255, 0.95);  /* 白色导航栏 */
  /* ... */
}
```

**建议方案**: 统一全站背景色风格，导航栏与内容区使用协调的配色方案。

---

### 3. 首页Hero区域文字颜色与背景冲突

**问题描述**:  
Home.vue多处使用白色文字（color: #fff, rgba(255,255,255,0.7)等），在style.css定义的浅色背景下完全不可见。

**具体位置**:
- 第343-344行: `.greeting-text h3 { color: #fff; }`
- 第350-351行: `.daily-quote { color: rgba(255, 255, 255, 0.7); }`
- 第455-456行: `.welcome-text h3 { color: #fff; }`
- 第460-461行: `.welcome-text p { color: rgba(255, 255, 255, 0.7); }`
- 第516行: `hero-title`使用包含白色的渐变
- 第523-524行: `.hero-subtitle { color: rgba(255, 255, 255, 0.7); }`
- 第570-571行: `.hero-hint { color: rgba(255, 255, 255, 0.5); }`

**建议方案**: 改为使用CSS变量 `var(--text-primary)` 和 `var(--text-secondary)`

---

### 4. 功能页面背景与主题冲突

**问题描述**:  
所有功能页面（Bazi.vue、Tarot.vue、Liuyao.vue、Hehun.vue、Daily.vue）使用深色背景，与style.css中白色主题定义不符。

**具体表现**:
- Bazi.vue: 加载状态使用 `background: linear-gradient(135deg, rgba(0, 0, 0, 0.3), rgba(0, 0, 0, 0.2))`
- 各页面卡片使用 `rgba(255,255,255,0.05)` 等深色半透明背景

**建议方案**: 统一页面背景配色，使用 `var(--bg-secondary)` 或 `var(--bg-card)`

---

### 5. 登录页深色背景与白色主题不协调

**问题描述**:  
Login.vue使用深色渐变背景（#1a1a2e到#16213e），与整体白色主题定义冲突。

**具体位置**: Login.vue第187行

```css
.login-page {
  background: linear-gradient(135deg, #1a1a2e 0%, #16213e 100%);
}
```

**建议方案**: 统一登录页主题风格，改为白色/浅色主题配色。

| Bazi.vue | 提示区域、按钮 | 💝、💎、🌱、🔮、❓、🎁、⚡ |

## 附录：设计规范建议

### 配色方案建议（如选择白色主题）

```css
:root {
  /* 主色调 */
  --primary-color: #e94560;
  --primary-light: #ff6b6b;
  --primary-gradient: linear-gradient(135deg, #e94560 0%, #ff6b6b 100%);
  
  /* 背景色 */
  --bg-primary: #ffffff;
  --bg-secondary: #f8f9fa;
  --bg-tertiary: #f0f2f5;
  --bg-card: #ffffff;
  
  /* 文字色 */
  --text-primary: #1a1a2e;
  --text-secondary: #4a5568;
  --text-tertiary: #718096;
  --text-muted: #a0aec0;
  
  /* 边框和阴影 */
  --border-color: #e2e8f0;
  --border-light: #edf2f7;
  --shadow-sm: 0 1px 3px rgba(0, 0, 0, 0.05);
  --shadow-md: 0 4px 6px rgba(0, 0, 0, 0.05);
  --shadow-lg: 0 10px 25px rgba(0, 0, 0, 0.08);
}
```
