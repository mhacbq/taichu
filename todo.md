# 太初命理 - UI 优化待办清单

## 🔴 P0 - 紧急修复（影响核心体验）

### [首页] 欢迎弹窗遮挡主内容
- **问题**：首次访问时出现"欢迎来到太初命理"引导弹窗，遮挡整个页面，用户无法直接看到产品内容
- **建议**：改为底部 Toast 提示或侧边滑入引导，不阻断主流程
- **文件**：`frontend/src/views/Index.vue`（引导组件相关逻辑）

### [全局] 后端 API 502 导致页面数据加载失败
- **问题**：多个页面出现 502 Bad Gateway 错误（统计数据、历史记录等），导致页面显示空白或错误状态
- **建议**：添加优雅降级处理，API 失败时显示友好的空状态 UI，而非空白
- **文件**：`frontend/src/api/index.js`、各页面组件

## 🔴 P1 - 高优先级（视觉体验明显缺陷）

### [首页] Hero 区域视觉权重不足
- **问题**：首页 Hero 区域高度偏小，主标题"欢迎来到太初命理"字体偏小，缺乏视觉冲击力
- **建议**：增大 Hero 区域最小高度（min-height: 60vh），主标题字号提升至 2.5-3rem，添加金色渐变装饰背景
- **文件**：`frontend/src/views/Index.vue`

### [首页] 功能卡片区域间距过于紧凑
- **问题**：桌面端功能卡片（八字排盘、塔罗占卜等）间距过小，视觉上显得拥挤
- **建议**：卡片间距增大至 24px，卡片内 padding 增加至 32px，添加 hover 上浮动效
- **文件**：`frontend/src/views/Index.vue`

### [八字排盘] 表单区域背景层次感不足
- **问题**：表单区域纯白背景，与页面整体缺乏层次区分，视觉平淡
- **建议**：表单容器添加淡金色背景（rgba(212, 175, 55, 0.05)）+ 金色细边框，增强卡片感
- **文件**：`frontend/src/views/Bazi.vue`（或拆分后的子组件）

### [八字合婚] 表单区域同上
- **问题**：男方/女方表单区域背景色过于平淡，缺乏层次感
- **建议**：同八字排盘，添加淡金色背景 + 细边框区分男女方区域
- **文件**：`frontend/src/views/Hehun.vue`

### [塔罗占卜] 牌面展示区域偏小
- **问题**：移动端塔罗牌展示区域高度不足，牌面图案显示不完整
- **建议**：移动端牌面最小高度设为 200px，确保牌面完整可见
- **文件**：`frontend/src/views/Tarot.vue`

## 🟡 P2 - 中优先级（体验优化）

### [全局导航] 移动端菜单展开体验
- **问题**：移动端汉堡菜单展开后，菜单项文字较小，点击区域偏小（不满足 44px 最小点击区域）
- **建议**：菜单项高度增至 48px，字号增至 16px，添加点击反馈动效
- **文件**：`frontend/src/components/Navbar.vue`（或 App.vue）

### [首页] 统计数字区域（用户数/占卜次数）
- **问题**：统计数字区域因 API 502 显示为 0 或空，即使有数据也缺乏视觉强调
- **建议**：数字使用大字号金色展示（2rem+），添加数字滚动动效，API 失败时显示"--"占位
- **文件**：`frontend/src/views/Index.vue`

### [六爻占卜] 卦象 SVG 展示区域
- **问题**：卦象 SVG 在移动端显示偏小，爻线细节不清晰
- **建议**：移动端 SVG 最小宽度 280px，动爻（红色）线条加粗至 3px
- **文件**：`frontend/src/views/Liuyao.vue`

### [每日运势] 幸运色预览圆点
- **问题**：幸运色颜色预览圆点尺寸偏小（约 12px），在移动端难以辨认
- **建议**：圆点尺寸增至 20px，添加颜色名称 tooltip
- **文件**：`frontend/src/views/Daily.vue`

### [八字排盘] "AI 深度分析"模块样式
- **问题**：AI 分析入口样式过于简单，像普通文字行，缺乏引导性
- **建议**：改为卡片式设计，添加 AI 图标 + 说明文字 + 积分消耗提示，金色渐变背景
- **文件**：`frontend/src/views/Bazi.vue`（或相关子组件）

### [全局] 空状态设计缺失
- **问题**：历史记录、分析结果等区域在无数据时显示空白，缺乏引导
- **建议**：统一空状态组件：图标 + 提示文字 + 操作按钮（如"开始第一次占卜"）
- **文件**：新建 `frontend/src/components/EmptyState.vue`

## 🟢 P3 - 低优先级（细节打磨）

### [全局] Footer 区域过于简单
- **问题**：Footer 仅有版权信息，缺乏品牌感
- **建议**：添加品牌 slogan（如"洞见命运，掌握人生"）、快速导航链接、社交媒体图标
- **文件**：`frontend/src/components/Footer.vue`（或 App.vue）

### [全局] 页面切换过渡动效
- **问题**：路由切换时无过渡动效，页面跳转生硬
- **建议**：添加 fade 过渡（opacity 0→1，duration 200ms）
- **文件**：`frontend/src/App.vue`（router-view transition）

### [八字合婚] 结果展示区域
- **问题**：合婚分析结果文字排版密集，缺乏视觉层次
- **建议**：结果分段展示，每段添加小标题 + 图标，关键评分用大字号金色突出
- **文件**：`frontend/src/views/Hehun.vue`

### [全局] 按钮 Loading 状态
- **问题**：点击"开始分析"等按钮后，Loading 状态不明显（仅文字变化）
- **建议**：按钮 Loading 时添加旋转图标 + 禁用状态样式，防止重复点击
- **文件**：各占卜页面组件
