# 太初命理 - 法律合规整改指南

## ⚠️ 重要声明

本指南旨在帮助网站符合国家法律法规要求，避免被认定为传播封建迷信。所有整改建议基于现行法律法规，但最终解释权归监管部门所有。

---

## 一、法律法规依据

### 1.1 核心法规

| 法规名称 | 关键条款 | 合规要点 |
|---------|---------|---------|
| 《互联网信息服务管理办法》第15条 | 禁止传播封建迷信 | 不得使用"算命""占卜"等表述 |
| 《网络安全法》 | 网络信息内容安全 | 需明确服务性质 |
| 《广告法》 | 不得含有迷信内容 | 不得承诺效果 |
| 《刑法》第300条 | 组织利用封建迷信犯罪 | 禁止经营性的迷信活动 |
| 2023"清朗"专项行动 | 打击违规算命服务 | 平台需自查整改 |

### 1.2 红线内容（绝对禁止）

```
❌ 绝对禁止：
├── 宣称"100%准确""必灵验"
├── 承诺"改运""转运""化解灾祸"
├── 提供"开光""做法事"等服务
├── 收费算命、占卜、看相
├── 预测具体事件（如死亡、疾病）
├── 恐吓用户"不化解就有灾祸"
└── 利用宗教名义敛财
```

## 三、文案整改方案

### 3.1 首页整改

**原文：**
```
八字排盘：精准的八字命理分析，解读您的人生轨迹
塔罗占卜：洞察未来可能
每日运势：趋吉避凶，把握良机
```

**整改后：**
```
八字分析：基于出生时间的性格特质分析
塔罗测试：趣味心理测试，探索内心世界
每日指南：今日幸运指数，娱乐参考
```

**原文：**
```
大运是十年一个周期的运势变化，影响人生不同阶段的吉凶祸福
流年是每年的运势变化，结合大运可精准预测年度吉凶
```

**整改后：**
```
大运是十年一个周期的人生阶段分析，反映不同时期的性格特点
流年是每年的运势参考，结合大运提供年度生活建议
```

### 3.3 塔罗页面整改

**原文：**
```
塔罗牌是一种心灵指引工具，可以帮助您了解当前状况
```

```
顺应变化，把握生活的机会
```

### 3.4 后端API整改

**原文代码：**
```php
protected function analyzeDaYunLuck(...) {
```

## 六、业务模式合规要点

### 6.1 收费模式

| 收费项目 | 合规性 | 建议 |
|---------|-------|-----|
| 八字分析收费 | ❌ 高风险 | 改为免费+广告模式 |
| 塔罗测试收费 | ❌ 高风险 | 改为免费+积分体系 |

| 周边商品 | ✅ 合规 | 可保留 |
| 心理咨询 | ⚠️ 需资质 | 如有需专业资质 |

### 6.2 积分体系合规

```
✅ 合规做法：
├── 积分用于解锁更多趣味测试
├── 积分通过签到、分享等获得
├── 积分不能兑换现金
└── 明确积分为虚拟道具

❌ 禁止做法：
├── 付费购买积分算命
├── 积分预测具体事件
└── 积分承诺准确结果
```

## 八、应急处理方案

### 8.1 被约谈/下架应急

```
第一步：立即下架敏感功能
- 关闭八字分析
- 关闭塔罗测试
- 保留纯文化介绍内容

第二步：整改声明
- 发布整改公告
- 明确整改方向
- 承诺合规运营

第三步：内容转型
- 转向纯文化科普
- 转向心理测试
- 增加娱乐属性
```

### 8.2 舆情危机处理

```
媒体曝光应对：
1. 第一时间回应，承认问题
2. 公布整改措施和时间表
3. 主动联系监管部门
4. 聘请法律顾问把关
5. 暂停争议功能
```
